# Complementing a Strand of DNA

**Problem ID:** REVC  
**Rosalind Link:** [REVC](https://rosalind.info/problems/revc/)

---

## Problem Description

Find the reverse complement of a DNA string.

### Input Format
A DNA string of length at most 1000 bp.

### Output Format
The reverse complement of the DNA string.

---

## Biological Context

DNA is double-stranded with complementary base pairing (A-T, C-G). The reverse complement is crucial in primer design, read alignment, and understanding DNA replication.

---

## Sample Dataset

**Input:**
```
AAAACCCGGT
```

**Output:**
```
ACCGGGTTTT
```
