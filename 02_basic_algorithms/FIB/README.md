# Rabbits and Recurrence Relations

**Problem ID:** FIB  
**Rosalind Link:** [FIB](https://rosalind.info/problems/fib/)

---

## Problem Description

Calculate the number of rabbit pairs after n months, assuming each pair produces k pairs per generation.

### Input Format
Two positive integers n and k (n ≤ 40, k ≤ 5).

### Output Format
The total number of rabbit pairs after n months.

---

## Biological Context

Models population growth under ideal conditions. Introduces recurrence relations, which appear in dynamic programming algorithms used in sequence alignment and gene prediction.
